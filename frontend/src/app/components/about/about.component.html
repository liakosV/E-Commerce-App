About This Project

GR

Αυτή η εφαρμογή δημιουργήθηκε ως μέρος της τελικής μου εργασίας για το πρόγραμμα του Coding Factory. Είναι η κορύφωση όλων όσων έμαθα — από την ανάπτυξη backend με Java, Spring Boot και βάσεις δεδομένων όπως το MySQL, μέχρι την κατασκευή ενός μοντέρνου και responsive frontend με Angular, Angular Material και Tailwind.

Το σύστημα περιλαμβάνει λειτουργίες όπως εγγραφή και σύνδεση χρηστών, περιήγηση προϊόντων με ταξινόμηση και φιλτράρισμα, καλάθι αγορών, δημιουργία παραγγελιών και διαχείριση από διαχειριστές για κατηγορίες, χρήστες, ρόλους, παραγγελίες και περιοχές. Υποστηρίζει επίσης έλεγχο πρόσβασης με ρόλους και ασφαλή αυθεντικοποίηση με JWT.

Αυτό το έργο όχι μόνο ενίσχυσε τις τεχνικές μου δεξιότητες, αλλά μου έδωσε και εμπειρία στην υλοποίηση μιας πραγματικής, full-stack web εφαρμογής. Από τον σχεδιασμό RESTful APIs και τη διαχείριση της ασφάλειας, μέχρι τη δημιουργία responsive UI components — κάθε μέρος της εφαρμογής χτίστηκε με καθαρό κώδικα, αρθρωτή δομή και επίκεντρο την εμπειρία του χρήστη.

Είμαι ιδιαίτερα περήφανος για το αποτέλεσμα και ευγνώμων για αυτό το ταξίδι. Οι εκπαιδευτές ήταν πάντα διαθέσιμοι να με βοηθήσουν με οποιαδήποτε απορία, και κάθε μάθημα ήταν γεμάτο με λεπτομερείς εξηγήσεις και πρακτικά παραδείγματα. Ευχαριστώ όλους — και ιδιαίτερα τους καθηγητές — που με στήριξαν καθ' όλη τη διάρκεια του προγράμματος!

EN

This application was built as part of my final project for the programm of Coding Factory. It is the culmination of everything I’ve learned — from backend development with Java and Spring Boot and Databases as MySQL to building a modern and responsive frontend using Angular, Angular Material and Tailwind.

The system includes features like user registration and login, product browsing with sorting and filtering, shopping cart functionality, order placement, and admin-level management of categories, users, and regions. It also supports role-based access control and secure JWT authentication.

This project not only helped solidify my technical skills, but also gave me experience in building a real-world, full-stack web application. From designing RESTful APIs and handling security to creating responsive UI components, every part of this app was built with clean code, modular structure, and user experience in mind.

I’m incredibly proud of the result and grateful for the journey. The instructors were always available to help with any question I had, and each class was packed with detailed explanations and practical examples. Thank you to everyone — especially the teachers — who supported me throughout the program!
